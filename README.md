# 📝 Notes App (Full Stack)

A simple full-stack Notes Application built using **Node.js, Express, MongoDB, and vanilla HTML/CSS/JS**.  
This app allows users to create, read, update, delete, and search notes. It is fully deployed and works on mobile and desktop.
